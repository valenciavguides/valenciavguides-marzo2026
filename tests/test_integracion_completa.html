Test Integración Completa

Resumen del Test de Integración

Este test valida que el índice de aventuras funciona correctamente con los datos reales y que los componentes pueden acceder a toda la información necesaria.

Componentes probados: Índice, Datos de coordenadas, Audios, Retos

Funcionalidades: Sincronización de datos, Conteos, Idiomas, Distribución
